1. Download and copy the provided files to your project folder:

  • If you have an empty repository, copy all the files
  • If you already have a working repository, copy the Dockerfile, the Jenkinsfile, and the 'deploy' folder.
  • Make sure that Flask, Flake8 and Pytest libraries are included in the requirements file
  • Modify the files deploy/lint.sh and deploy/test.sh whether further customizations are required for linting/testing

2. Add the new files to Git and commit your changes to the repository.

3. Go to Jenkins to checkout your pipelines.
